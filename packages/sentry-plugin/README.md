# Deenruv Sentry Plugin

Integrates your Deenruv server with the [Sentry](https://sentry.io/) application monitoring service.

`npm install @deenruv/sentry-plugin`

For documentation, see [docs.deenruv.io/reference/core-plugins/sentry-plugin/](https://docs.deenruv.io/reference/core-plugins/sentry-plugin/)
