# CustomCardHeader

A default Deenruv card header component. In most cases it shouldn't be user separately.

## Props

### icon

- **Type:** `ReactNode`

- An optional icon displayed alongside the title.

### title

- **Type:** `string`

- The title of the card.

### description

- **Type:** `string`

- An optional description displayed below the title.

### isCollapsed

- **Type:** `boolean`

- If true, the description is truncated.

### fullWidth

- **Type:** `boolean`

- If true, sets the wrapper to full width.

### children

- **Type:** `ReactNode`

- An optional element positioned in the upper-right corner.



## Example Usage

```tsx
<CustomCardHeader
  icon={/* value */}
  title={/* value */}
  description={/* value */}
  isCollapsed={/* value */}
  fullWidth={/* value */}
  children={/* value */}
/>
```
