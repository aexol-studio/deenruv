# ConfirmationDialog

A modal that asks the user to confirm an action.

## Props

### onConfirm

- **Type:** `() => void`

- Callback invoked when the user confirms the action.

### title

- **Type:** `string`

- The title of the dialog.

### description

- **Type:** `string`

- The description of the dialog.

### children

- **Type:** `ReactNode`

- The trigger element that opens the dialog.



## Example Usage

```tsx
<ConfirmationDialog
  onConfirm={() => {}}
  title={/* value */}
  description={/* value */}
  children={/* value */}
/>
```
